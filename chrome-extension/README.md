# GTD Timer Chrome Extension

A simple 2-minute productivity timer for focused work sessions, packaged as a Chrome extension.

## Installation

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" in the top right
3. Click "Load unpacked" and select the `chrome-extension` folder
4. The GTD Timer extension will appear in your toolbar

## Usage

- Click the extension icon in your toolbar to open the timer
- Press the play button to start a 2-minute focus session
- The timer continues running even when the popup is closed
- You'll receive a notification when each 2-minute session completes
- The timer automatically resets for the next session

## Features

- **Background operation**: Timer continues when popup is closed
- **Persistent state**: Timer state is saved across browser sessions
- **Notifications**: Desktop notifications when sessions complete
- **Cycle tracking**: Keeps count of completed sessions
- **Clean interface**: Minimal design optimized for the extension popup

## Icons

To complete the installation, you'll need to add icon files to the `icons/` directory:
- `icon16.png` (16x16 pixels)
- `icon32.png` (32x32 pixels) 
- `icon48.png` (48x48 pixels)
- `icon128.png` (128x128 pixels)
